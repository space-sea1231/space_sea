# CF131D Subway

## 题目描述

给出一个 $n$ 个点，$n$ 条边的无向无权图，图上每条边的长度为 $1$，保证图中有且仅由一个环。

你的任务是求出每一个点到环（环上任意一点）的最短路径长度。

## 输入格式

第一行一个整数 $n$（$3 \leq n \leq 3 \times 10^3$），表示点的个数和边的个数。

接下来 $n$ 行，每行两个整数 $u,v$，表示 $u,v$ 之间有一条边，$1 \le u,v \le n$。

## 输出格式

一行 $n$ 个由空格隔开的整数，表示每个点到环的最短距离。

## 输入输出样例 #1

### 输入 #1

```
4
1 3
4 3
4 2
1 2
```

### 输出 #1

```
0 0 0 0
```

## 输入输出样例 #2

### 输入 #2

```
6
1 2
3 4
6 4
2 3
1 3
3 5
```

### 输出 #2

```
0 0 0 1 1 2
```
